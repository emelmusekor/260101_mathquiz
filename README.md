# 초등학생용 감성 수학 퀴즈 앱

초등학생(3~4학년)을 대상으로 한 감성 커스텀 수학 퀴즈 애플리케이션입니다. 16가지 감정 중 하나를 선택하면 위로가 되는 명언을 보여주고, 학생의 응답 수준에 따라 난이도가 자동 조절되는 시스템을 갖추고 있습니다.

## 버저닝 및 업데이트 이력 (Changelog)

### v1.1 (최신 버전)
- **데이터베이스 분리 구조 적용:** `app.js`에 있던 정적 변수들을 `database` 폴더 하위의 용도별 자바스크립트 파일(`moods.js`, `questions_low.js`, `questions_mid.js`, `questions_high.js`)로 완전히 분리.
- **감정 16종 세분화:** 16가지의 감정별로 약 5개씩 총 80개의 서로 다른 명언 데이터를 구축하고 무작위 출력.
- **초등수학 동적 난이도 조절 시스템 (Dynamic Difficulty):** 
  - 기본 시작 난이도는 (중).
  - 오답 시 즉각적으로 (하) 난이도로 다운그레이드. 맞출 시 (상) 난이도로 업그레이드 지원.
  - 연산 종류: 덧셈, 뺄셈, 곱셈(구구단), 나눗셈. 
- **연속 3번 정답 로직:** 목표치인 '3회 연속 정답' 달성 시 즉시 성공 화면으로 이동.
- **종합 해설 제공:** 최종 성공 화면에서 사용자가 푼 모든 문제들의 정답 및 풀이 과정 일괄 확인 지원.
- **객관식 및 주관식(단답형) 혼용 지원.**

### v1.0 (초기 릴리스)
- 싱글 페이지 HTML/CSS/JS 뼈대 구현.
- 4가지 기분에 대한 정적 하드코딩 피드백 및 단순 퀴즈 로직 포함.
- 다크모드 및 Glassmorphism 반응형 CSS 적용.

## 실행 방법
`index.html` 파일을 더블 클릭하여 실행합니다. 데이터베이스는 `.js` 파일로 브라우저 콘텍스트 내부에 로드되므로, 별도로 Live Server 등의 가상 환경 없이 완벽하게 오프라인에서도 동작합니다.
