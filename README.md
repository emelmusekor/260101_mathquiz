# 감성 수학 퀴즈 앱 (채팅 연동 버전)

초등학생(3~4학년)을 대상으로 한 수학 퀴즈 애플리케이션에 실시간 좌측 사이드바 채팅 기능이 포함된 최신 버전입니다. 사용자의 기분에 공감하고 문제 해결 능력에 따라 난이도가 조절되는 동적 반응형 웹앱입니다.

## 버저닝 및 업데이트 이력 (Changelog)

### v1.3 (최신 버전 - 실시간 채팅 추가)
- **Supabase 백엔드 연동:** 실시간 채팅 모듈(`chat.js`)을 추가하여 다중 접속 환경 관리 및 실시간 동기화 구현. 
- **레이아웃 구조 개편:** 기존 중앙 집중식 글래스모피즘 카드 뷰를 해제하고, [좌측 채팅 사이드바] + [우측 메인 퀴즈부] 형태의 2단 분할 레이아웃 적용. 
- **모바일 대응 개선:** 화면이 좁아지는 스마트폰 기기에서는 우측 퀴즈화면이 우선 노출되고 채팅창이 스크롤 아래에 붙도록 Flex 레이아웃 스위칭.

### v1.2 (대규모 10단계 DB 업데이트)
- 자동화 스크립트(`generate_db.py`)를 이용한 1,000만 제 DB 생성 및 10단계 초정밀 난이도 그래프 적용.
- 성공, 오답, 콤보 상승 시 다양한 시각효과(VFX) 트리거. 
- 처음 접속 시 고른 감정 데이터를 기억하여 성공 화면에서 위로/응원 메시지로 응답.

## 🚀 실행 및 채팅(Supabase) 설정 방법 안내

`index.html`을 브라우저에서 실행하기만 해도 디자인 레이아웃을 모두 보실 수 있습니다. 하지만 **실시간 채팅 기능을 온전히 구동하시려면 다음의 2단계가 반드시 선행**되어야 합니다.

### 1단계: API Key 연동
1. 다운받으신 폴더 안의 `chat.js` 파일을 메모장이나 VS Code 에디터로 엽니다.
2. 최상단(8~9번째 줄)의 주석 가이드를 읽고, 발급받으신 **SUPABASE_URL** 과 **SUPABASE_KEY** (Anon/Public)를 정확한 자리에 붙여넣으세요.

### 2단계: 데이터베이스 테이블 및 권한 생성
현재 구현된 코드에 맞게 통신하려면 Supabase 대시보드(SQL Editor)에서 아래 명령어를 쳐서 테이블과 권한(RLS)을 생성해주셔야 합니다.

```sql
-- 테이블은 이미 만드셨으므로 권한 추가만 진행하시면 됩니다.

-- 1. 실시간 기능(Realtime) 활성화
alter publication supabase_realtime add table chat_server;

-- 2. 익명 참가자의 쓰기/가져오기 권한(RLS) 허용
alter table chat_server enable row level security;
create policy "누구나 읽기 가능" on chat_server for select using (true);
create policy "누구나 전송 가능" on chat_server for insert with check (true);
```
위 설정을 마치면 좌측 사이드바에서 닉네임을 치고 자유롭게 실시간 통신을 테스트해 보실 수 있습니다!
